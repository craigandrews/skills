# Agent Skills: craigandrews/skills

Reusable agent skills for common engineering workflows.

## Install

```bash
npx skills add craigandrews/skills
```

## Available skills

- `python-venv` in `skills/venv/SKILL.md`

## Contributing

- Add new skills under `skills/<skill-name>/SKILL.md`
- Include YAML frontmatter with `name` and `description`
